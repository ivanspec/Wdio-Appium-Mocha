# Wdio-Appium-Mocha

## Requirement 
- Node JS (https://nodejs.org/en/download/)
- Appium Server (npm install -g appium) / (https://github.com/appium/appium-desktop/releases/tag/v1.22.2)
- IDE Editor (VSCode / Intelij
- Android Studio / SDK
- JDK_8
